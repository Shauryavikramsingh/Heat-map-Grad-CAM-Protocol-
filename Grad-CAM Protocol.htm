Grad-CAM Protocol

Turning "Black-Box" Predictions into Visual Medical Evidence.

The "Why" Behind the Code

In medical diagnostics, knowing "What" the AI predicted isn't enough. We must know "Where" the AI looked. Most students use simple predictions; we implement Gradient-weighted Class Activation Mapping.

1. Gradient Tracking: We use tf.GradientTape() to monitor the "Pneumonia" score as it flows through the neural network.
2. Feature Importance: We look at the Last Convolutional Layer. This layer contains the most complex spatial patterns (like lung textures).
3. Pixel Activation: By calculating the gradient of the score with respect to these features, we find exactly which pixels "pushed" the AI toward its decision.

The Lethal Code

import tensorflow as tf
import numpy as np

def generate_medical_heatmap(model, img_array, layer_name):
    # 1. Map input to the specific feature layer
    grad_model = tf.keras.models.Model(
        [model.inputs], [model.get_layer(layer_name).output, model.output]
    )

    # 2. Record the gradients of the top predicted class
    with tf.GradientTape() as tape:
        conv_outputs, predictions = grad_model(img_array)
        class_idx = np.argmax(predictions[0])
        loss = predictions[:, class_idx]

    # 3. Compute the weights of each feature map
    output = conv_outputs[0]
    grads = tape.gradient(loss, conv_outputs)[0]
    weights = tf.reduce_mean(grads, axis=(0, 1))

    # 4. Create the heatmap (weighted average)
    heatmap = output @ weights[..., tf.newaxis]
    heatmap = tf.squeeze(heatmap)
    return heatmap.numpy()
